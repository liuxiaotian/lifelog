- **Primary**: Amber (#FFC107)
- **Background**: Amber 50 with 30% opacity
- **Glow**: Amber with 50% opacity, 8px blur
- **Border**: 3px solid amber

### Map Marker Colors
- **Regular Entry**: Blue (#2196F3)
- **Highlighted Entry**: Amber (#FFC107) with glow
- **Background**: Light blue/amber based on type

## User Flow

### Creating a Highlight
1. User taps "+" button
2. Fills in entry details
3. Checks "Mark as Highlight" âœ…
4. Saves entry
5. Entry appears with golden styling on timeline

### Viewing All Highlights
1. User opens Settings
2. Taps "Life Highlights"
3. Sees all highlighted entries in list
4. Can tap any to view full details

### Viewing Event Map
1. User opens Settings
2. Taps "Event Map"
3. Map loads with all entries that have locations
4. Highlighted entries show golden markers
5. Tap marker to see entry details

### Changing Time Format
1. User opens Settings
2. Scrolls to "Time Format"
3. Selects preferred format
4. Returns to timeline - sees new format
5. Format persists across app restarts

## Accessibility

- All interactive elements have proper tap targets (48x48 dp minimum)
- Color is not the only indicator (star icons used)
- Text maintains sufficient contrast
- Screen reader compatible labels
- Keyboard navigation support (Flutter default)

## Responsive Design

- Works in portrait and landscape modes
- Timeline adapts to orientation (existing feature)
- Map fills available space
- Settings scrollable for all screen sizes
- Cards have minimum/maximum widths
